# Laravel PHP Bootstrap CSS CRUD Application

This project is a CRUD (Create, Read, Update, Delete) application built using Laravel PHP framework with Bootstrap CSS for styling and MySQL database for data storage.

## Prerequisites

Before running this application locally, ensure you have the following installed:

- [XAMPP](https://www.apachefriends.org/index.html) or similar software for local development environment.
- PHP (version >= 7.4.5) installed.
- Composer (for installing PHP dependencies).
- MySQL (via XAMPP or other setup).
- Laravel Framework (version 8.83.27).
- VS Code (IDE)

## Installation

1. Unzip the file
2. Open the php_Crud_Application file using VS Code
3. Create Database using provide invetorysystem.sql document(Databse name = `invetorysystem`)
4. Serve the application locally (using XAMPP)
5. Access the application in your web browser at `http://localhost:8000`.

## Usage

- The application allows you to perform CRUD operations on items stored in the database.
- Navigate to different routes defined in `routes/web.php` to view, add, update, or delete items.
- Ensure you have XAMPP running with Apache and MySQL services before starting the application.

## Technologies Used

- **Framework**: Laravel PHP Framework (v8.83.27)
- **Styling**: Bootstrap CSS
- **Database**: MySQL (via XAMPP)
- **Development Tools**: Composer, PHP (v7.4.5)
